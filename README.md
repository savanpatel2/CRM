# CRM Sales Pipeline Analytics Dashboard (Power BI)

## Overview
This repository contains a Power BI project designed to analyze and visualize CRM sales pipeline data. The dashboard provides an executive-style view of sales performance, tracking key metrics, pipeline progression, and goal attainment to assist in strategic decision-making[cite: 1].

## Features
* **Pipeline Analytics**: Comprehensive insights into sales stages, deal values, and conversion rates[cite: 1].
* **Executive KPIs**: Key performance indicators (KPIs) highlighting total profit, performance metrics, and progress against sales targets[cite: 1].
* **Data-Driven Modeling**: Utilizes a robust **Star Schema** data model to ensure accurate relationships and optimized reporting performance[cite: 1].
* **Custom UI/UX**: Built with a modern aesthetic, including a consistent color palette and custom branding (e.g., 'PEAK PERFORM ANALYTICS' logo) for a professional executive presentation[cite: 1].

## Prerequisites
* **Microsoft Power BI Desktop** (Latest version recommended)
* **Python** (Required if performing additional data cleaning or merging on the source dataset)[cite: 1]
* Access to the underlying CRM dataset (e.g., SQL database, CSV, or Excel file)

## Installation & Setup
1. **Clone the repository**: Download or clone the project files to your local machine[cite: 1].
2. **Open in Power BI**: Load the `CRM.pbix` file into Power BI Desktop[cite: 1].
3. **Data Processing**: If the source data requires updates, ensure your Python environment is configured to clean and merge the dataset as per the project requirements[cite: 1].
4. **Refresh Data**: Once connected to your local data source, refresh the data model to populate the dashboard visuals[cite: 1].

## Repository Structure
* **CRM.pbix**: The core Power BI file containing the data model, DAX measures, relationships, and the executive-style dashboard layout[cite: 1].
* **README.md**: Project documentation and setup instructions.

*Note: Ensure your source data schema matches the schema expected by the dashboard visuals to maintain reporting accuracy.*
